# 🚀 Projeto ETL TBH Esportes

---

# 📌 Objetivo do Projeto

Este projeto tem como objetivo demonstrar, através de um case real, minhas habilidades na área de dados aplicadas em um cenário completo de engenharia e análise de dados, utilizando também recursos de Inteligência Artificial para auxiliar no processo de captura e estruturação das informações.

A proposta consiste na construção de uma pipeline de dados ponta a ponta, passando pelas etapas de:

- Extração dos dados
- Tratamento e padronização
- Estruturação analítica
- Armazenamento em nuvem
- Modelagem de dados
- Criação de dashboards no Power BI

Os dados utilizados neste projeto são provenientes dos resultados de corridas disponibilizados em arquivos PDF. Parte da captura e interpretação dessas informações contou com o auxílio de Inteligência Artificial, tornando possível acelerar o processo de transformação de dados não estruturados em uma base analítica organizada.

---

# 🏃 Possibilidades de Análise

Com a construção dessa base de dados, torna-se possível desenvolver diferentes visões analíticas voltadas ao universo da corrida.

## 👟 Perfil do Corredor

Permite acompanhar a evolução individual do atleta ao longo do ano, possibilitando análises como:

- Evolução de performance
- Comparativo de tempos
- Frequência de participação
- Distâncias percorridas
- Histórico de provas
- Ritmo médio e evolução esportiva

---

## 📊 Perfil de Assessoria de Corrida

Voltado para assessorias esportivas que desejam acompanhar o desempenho de seus atletas de forma estratégica e individualizada.

Com essa visão, é possível:

- Monitorar evolução dos atletas
- Identificar quedas ou melhorias de performance
- Comparar desempenho entre corredores
- Acompanhar participação em eventos
- Gerar insights para consultoria personalizada
- Apoiar decisões técnicas com dados
